# Guide d'Utilisation de l'Application de Gestion de Bibliothèque

Bienvenue dans l'application de gestion de bibliothèque ! Ce guide vous expliquera comment utiliser les fonctionnalités de l'application pour gérer efficacement votre collection de livres.

## Démarrez l'application.

## Ajouter un Livre

Pour ajouter un livre à votre bibliothèque, suivez ces étapes :
1. Sélectionnez l'option "Ajouter un Livre".
2. Suivez les instructions pour saisir les détails du livre, tels que le titre, l'auteur et le numéro ISBN.
3. Le livre sera ajouté à votre bibliothèque avec le statut "Disponible".

## Afficher la Liste des Livres Disponibles

Pour afficher la liste des livres disponibles dans votre bibliothèque, faites ce qui suit :
1. Sélectionnez l'option "Afficher la Liste des Livres Disponibles".
2. La liste des livres disponibles sera affichée à l'écran.

## Rechercher un Livre

Pour rechercher un livre dans votre bibliothèque, procédez comme suit :
1. Sélectionnez l'option "Rechercher un Livre".
2. Saisissez le titre ou l'auteur du livre que vous recherchez.
3. Les résultats de la recherche seront affichés, le cas échéant.

## Emprunter un Livre

Si vous souhaitez emprunter un livre, suivez ces étapes :
1. Sélectionnez l'option "Emprunter un Livre".
2. Saisissez le numéro ISBN du livre que vous souhaitez emprunter.
3. Si le livre est disponible, saisissez votre nom d'utilisateur et la date de retour prévue.
4. Le livre sera marqué comme "Emprunté" et sera retiré de la liste des livres disponibles.

## Retourner un Livre

Pour retourner un livre, effectuez les étapes suivantes :
1. Sélectionnez l'option "Retourner un Livre".
2. Saisissez le numéro ISBN du livre que vous souhaitez retourner.
3. Le livre sera marqué comme "Disponible" et ajouté à la liste des livres disponibles.

## Supprimer un Livre

Pour supprimer un livre de votre bibliothèque, suivez ces étapes :
1. Sélectionnez l'option "Supprimer un Livre".
2. Saisissez le numéro ISBN du livre que vous souhaitez supprimer.
3. Le livre sera supprimé de votre bibliothèque.

## Modifier les Informations d'un Livre

Si vous souhaitez modifier les informations d'un livre, procédez comme suit :
1. Démarrez l'application.
2. Sélectionnez l'option "Modifier les Informations d'un Livre".
3. Saisissez le numéro ISBN du livre que vous souhaitez modifier.
4. Suivez les instructions pour mettre à jour les détails du livre.

## Générer un Rapport Statistique

Pour générer un rapport statistique de votre bibliothèque, suivez ces étapes :
1. Sélectionnez l'option "Générer un Rapport Statistique".
2. Un rapport détaillé sur votre bibliothèque sera affiché.

C'est tout ! Vous savez maintenant comment utiliser notre application de gestion de bibliothèque. Profitez de la gestion facile de vos livres !


---
**Note :** Assurez-vous d'avoir une base de données MySQL configurée et connectée à l'application avant d'utiliser les fonctionnalités .
